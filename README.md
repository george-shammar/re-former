### Usage

- Create new folder.
- Open your terminal.
- Run ``` git clone https://github.com/pipe2442/re-former.git ```.
- Enter root folder with command ``` cd re-former ```.
- In the terminal from the root folder, type ``` rails s ``` and run the browser to display the project.
